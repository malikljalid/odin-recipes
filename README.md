# odin-recipes
In this project, i am going to build a basic recipe webpage.

The webpage will consist of a main index page which will have links to a few recipes.
In order to achieve that, i will use HTML tags and elements only without any styling.

I made recipes of 3 dished : Chili, Pizza and Tacos,

I provided clear description of each recipe by including two paragraphs in each recipe page,

I provided unordered list of ingredients for each recipe,

I provided ordered list of steps to follow to make each recipe,

I provided an image of each recipe,

I linked the main page to each recipe by using an unordered list, and vesca-versa for each recipe page.
